Rust Programming Language

Introduction and History

Rust is a modern systems programming language developed by Mozilla Research. Its first stable release was in 2015. It focuses on safety, concurrency, and performance without a garbage collector. Rust is used by companies like Dropbox, Cloudflare, and Microsoft.

Installation Instructions

curl https://sh.rustup.rs -sSf | sh

After installing, run rustc --version to verify.

Basic Examples

Hello, World

fn main() {
    println!("Hello, world!");
}

Build/Run:

rustc main.rs
./main

Fibonacci Generator

fn fib(n: u32) -> u32 {
    match n {
        0 => 0,
        1 => 1,
        _ => fib(n - 1) + fib(n - 2),
    }
}

Advanced Example

Multithreaded Web Server (Simple)

use std::net::TcpListener;
use std::thread;

fn main() {
    let listener = TcpListener::bind("127.0.0.1:7878").unwrap();
    for stream in listener.incoming() {
        thread::spawn(|| {
            let _ = stream.unwrap();
            println!("Connection received!");
        });
    }
}
